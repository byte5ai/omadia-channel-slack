So richtest du die Slack-App ein (api.slack.com/apps):

  1. OAuth & Permissions → Bot-Scopes setzen (app_mentions:read, chat:write, reactions:write, im:history, channels:history, users:read) → App installieren → Bot-Token (xoxb-…).
  2. Events API: Signing Secret (Basic Information) hinterlegen, dann unter Event Subscriptions die Request URL oben eintragen und die Bot-Events app_mention, message.im (opt. message.channels) abonnieren.
  3. Socket Mode: Socket Mode aktivieren + App-Level-Token mit connections:write (xapp-…) hinterlegen.
  4. Bot in die gewünschten Channels einladen (/invite @bot).
