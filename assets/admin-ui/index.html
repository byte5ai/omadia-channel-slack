So richtest du die Slack-App ein:

  1. Auf api.slack.com/apps eine App erstellen.
  2. Socket Mode aktivieren und einen App-Level-Token mit Scope connections:write erzeugen (xapp-…).
  3. Unter OAuth & Permissions die Bot-Scopes setzen (app_mentions:read, chat:write, reactions:write, im:history, channels:history, users:read) und die App installieren → Bot-Token (xoxb-…).
  4. Unter Event Subscriptions die Bot-Events app_mention, message.im (und optional message.channels) abonnieren.
  5. Beide Tokens in den Plugin-Einstellungen hinterlegen.
